# _ESP32 Camera QR Decode_

Capture QR code via ESP-CAM (or a connected ov5640) and Send result to UART port


## How to use
Just hit idf.py build to import all required files including managed components.

## Example folder contents

The project **ESP32-Camera-QR-Decode** contains one source file in C language [main.c](main/main.c). The file is located in folder [main](main).
